# 🎮 Discord Online Players Bot

Бот для Discord который автоматически отслеживает и отображает количество онлайн игроков в выбранном канале.

## ✨ Возможности

- ✅ Автоматическое создание канала для отслеживания
- ✅ **Редактирование** сообщения вместо создания новых
- ✅ Отображение количества онлайн игроков
- ✅ Список активных игроков
- ✅ Автоматическое обновление при изменении статуса

## 📋 Требования

- Python 3.8+
- Discord.py 2.3.2+
- Токен Discord бота

## 🚀 Установка и запуск

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/ffff55567/portfolio.git
cd portfolio
```

### 2. Создайте виртуальное окружение

```bash
# Linux/Mac
python3 -m venv venv
source venv/bin/activate

# Windows
python -m venv venv
venv\Scripts\activate
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Получите токен бота

1. Зайдите в [Discord Developer Portal](https://discord.com/developers/applications)
2. Нажмите "New Application"
3. Назовите приложение
4. Перейдите на вкладку "Bot"
5. Нажмите "Add Bot"
6. Нажмите "Reset Token" и скопируйте токен

### 5. Создайте файл `.env`

```bash
cp .env.example .env
```

Откройте `.env` и замените `YOUR_TOKEN_HERE` на ваш реальный токен:

```
DISCORD_TOKEN=ваш_токен_здесь
```

### 6. Дайте боту необходимые права

В Discord Developer Portal → Bot → Permissions выберите:
- ✅ Send Messages
- ✅ Read Messages/View Channels
- ✅ Manage Channels
- ✅ Embed Links

### 7. Добавьте бота на сервер

В Discord Developer Portal → OAuth2 → URL Generator выберите:
- Scopes: `bot`
- Permissions: `Send Messages`, `Manage Channels`, `Read Messages`

Скопируйте созданную ссылку и откройте её в браузере.

### 8. Запустите бота

```bash
python discord_bot.py
```

Вы должны увидеть:
```
✅ Бот YourBotName подключен!
```

## 📖 Команды

### `!create_status`
Создает новый канал для отслеживания онлайн игроков.

```
!create_status
```

### `!status`
Показывает текущий статус онлайн игроков на сервере.

```
!status
```

## 🐛 Решение проблем

### Ошибка: "Token is invalid"
- Проверьте, что вы скопировали токен правильно
- Убедитесь, что файл `.env` находится в корне проекта

### Бот не реагирует на команды
- Убедитесь, что боту даны права на отправку сообщений
- Проверьте, что префикс команды `!` правильный

### Канал не создается
- Убедитесь, что боту даны права на управление каналами
- Проверьте права в Discord сервере
