# Портфолио - Web разработчик

Это красивое и современное портфолио веб-разработчика, созданное с использованием React, TypeScript и Framer Motion.

## 🌟 Особенности

- ✨ Современный, привлекательный дизайн с градиентами и анимациями
- 📱 Полная адаптивность (работает на всех устройствах)
- ⚡ Быстрая производительность
- 🎭 Плавные анимации и переходы
- 📊 Проекты с описаниями и ссылками
- 🎯 Секции: О мне, Проекты, Навыки, Контакты
- 🌓 Красивый UI с современным стилем

## 🛠️ Технологический стек

- **React 18** - UI библиотека
- **TypeScript** - Типизация JavaScript
- **Vite** - Быстрый сборщик проектов
- **Framer Motion** - Анимации
- **React Icons** - Иконки
- **CSS 3** - Стилизация

## 📦 Установка

```bash
# Клонируем репозиторий
git clone https://github.com/ffff55567/portfolio.git
cd portfolio

# Устанавливаем зависимости
npm install

# Запускаем проект локально
npm run dev

# Собираем для продакшена
npm run build
```

## 🚀 Развертывание

### GitHub Pages

1. Создайте ветку `gh-pages`
2. Обновите `vite.config.ts` с правильным `base`
3. Запустите `npm run build`
4. Загрузьте содержимое `dist` на `gh-pages`

### Vercel / Netlify

1. Подключите репозиторий
2. Выберите фреймворк: Vite
3. Команда сборки: `npm run build`
4. Директория публикации: `dist`

## 📝 Кастомизация

Для изменения контента отредактируйте:
- `src/components/About.tsx` - Информацию о себе
- `src/components/Projects.tsx` - Ваши проекты
- `src/components/Skills.tsx` - Ваши навыки
- `src/components/Contact.tsx` - Контактную информацию
